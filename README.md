# Tom Groves Static Site

This is a dependency-free static site.

Open `index.html` directly in a browser, or upload the folder contents to any
static host such as S3, Google Cloud Storage, Cloudflare Pages, Netlify or
Vercel.

## Cheapest Hosting Shape

- Store `index.html`, `styles.css` and `assets/tom-groves.jpg` in a public
  static bucket.
- Put CloudFront or Cloudflare in front of it if you want HTTPS and caching.
- No server, database, build step or runtime is required.
